## ⚙️ Visão Geral do Pipeline de Dados

Este projeto foi desenvolvido para pôr em prática ensinos do primeiro módulo do Curso Técnico em IA do IF.
Aqui apliquei um pipeline completo de Ciência de Dados utilizando Python, com o objetivo de prever se uma faixa musical será um "Hit" (Sucesso) ou um "Flop" (Fracasso) com base em seus atributos de áudio. A base de dados escolhida proveniente do Kaggle possui mais de 113.000 linhas e era bem avaliada para uso em projetos assim. Em cada etapa do pipeline eu deixei anotações explicativas e notas que evidenciam os motivos de minhas decisões e conclusões.
